# Lancements

Les lancements en AEM Sites permettent de créer, d’écrire et de consulter le contenu d’un site Web en vue d’une prochaine version. Pendant la création du lancement, le site Web de production peut continuer à évoluer et à changer au jour le jour comme il le ferait normalement.
