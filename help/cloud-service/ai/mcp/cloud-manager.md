# Serveur MCP Cloud Manager

Le serveur MCP Cloud Manager permet aux développeurs AEM d’exécuter des pipelines, de déboguer des échecs et d’accéder directement aux outils Cloud Manager dans leur IDE, sans changement de contexte.

>[!AVAILABILITY]
>
> Le serveur Cloud Manager MCP est désormais en version bêta. Pour demander l’accès, envoyez un e-mail à [aemcs-mcp-feedback@adobe.com](mailto:aemcs-mcp-feedback@adobe.com) avec une description de votre cas d’utilisation.
