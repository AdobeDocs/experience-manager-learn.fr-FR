# Configurer l’IA dans AEM

Pour utiliser l’assistant et les agents AEM AI dans AEM, vous devez configurer l’accès dans votre environnement Adobe Experience Manager. Ce guide vous guidera à travers les étapes nécessaires pour activer ces fonctionnalités optimisées par l’IA. Vous pouvez également tester l’assistant d’IA dans un environnement de jeu.

## Assistant IA AEM

Découvrez comment configurer l’accès à l’assistant AI dans Adobe Experience Manager à l’aide de [Adobe Admin Console](https://adminconsole.adobe.com). Cette étape est nécessaire pour permettre aux utilisateurs d&#39;obtenir des réponses de l&#39;assistant et d&#39;utiliser la création automatique de tickets.

>[!VIDEO](https://video.tv.adobe.com/v/3474066/?learn=on&enablevpops)

### Agents dans AEM

Pour utiliser les agents dans AEM, vous devez d&#39;abord avoir configuré l&#39;accès à [l&#39;assistant IA d&#39;AEM](#aem-ai-assistant). Les agents dans AEM doivent être configurés par Adobe. Pour demander l’accès aux agents dans AEM, contactez votre représentant Adobe ou [l’assistance Adobe](https://experienceleague.adobe.com/en/support).

## Laboratoire

Si vous n’avez pas accès à l’assistant AEM AI ou aux agents dans AEM, ou si vous ne pouvez pas les configurer, vous pouvez tout de même explorer leurs fonctionnalités dans un environnement de jeu.

AEM fournit un environnement de jeu où les utilisateurs peuvent tester les fonctionnalités de l’assistant AI, y compris les agents dans AEM, sans affecter leur environnement de production. Cela permet aux utilisateurs de se familiariser avec les fonctionnalités de l’assistant d’IA dans un environnement sûr.

![Terrain de jeu AEM](./assets/setup/playground.png)

Pour créer un terrain de jeu, accédez à [Experience Hub](https://experience.adobe.com/#/@AdobeOrg/experiencemanager/) et sélectionnez **Créer un terrain de jeu** dans le panneau **Annonces de produits**. Une fois votre terrain de jeu configuré, vous recevrez un e-mail contenant les détails pour y accéder.
