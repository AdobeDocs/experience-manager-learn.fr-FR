# L’IA dans Experience Manager

![AI dans Experience Manager](./assets/overview/hero.png)

Experience Manager as a Cloud Service fournit des fonctionnalités d’IA avancées pour améliorer la gestion de contenu, rationaliser les workflows et améliorer les expériences utilisateur. En intégrant des fonctionnalités basées sur l’IA telles que l’assistant AEM AI et les agents AEM AI, les utilisateurs et utilisatrices peuvent automatiser les tâches, obtenir des informations et optimiser la diffusion de contenu.
