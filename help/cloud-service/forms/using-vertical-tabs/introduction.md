# Présentation

Les onglets verticaux permettent d’organiser et d’afficher le contenu dans un format compact et facile à parcourir. Vertical offre les avantages suivants
1. Efficacité de l’espace : les onglets verticaux utilisent l’espace vertical, qui peut être plus efficace lorsqu’il y a beaucoup d’informations à saisir.
1. Navigation améliorée : ils permettent aux utilisateurs de basculer rapidement entre les différentes sections d’un formulaire.
1. Clarité visuelle : ils permettent de maintenir une mise en page nette en conservant les options de navigation à un seul endroit.
