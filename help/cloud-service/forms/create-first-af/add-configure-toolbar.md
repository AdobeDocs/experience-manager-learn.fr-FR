# Ajouter et configurer la barre d’outils

La barre d’outils est généralement ajoutée au niveau du conteneur du formulaire. La barre d’outils peut contenir des boutons pour accéder aux panneaux précédent et suivant de votre formulaire, envoyer le formulaire, enregistrer les données du formulaire, etc.
