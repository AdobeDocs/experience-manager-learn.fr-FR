# Intégration d’AEM Forms à Adobe Analytics pour créer des rapports sur les champs de données de formulaire

Découvrez comment intégrer AEM Forms as a Cloud Service à Adobe Analytics sur vos formulaires adaptatifs à l’aide de balises Experience Platform. Cet exemple vous guidera tout au long des étapes de configuration et d’implémentation pour produire des rapports pertinents sur la manière dont les visiteurs interagissent avec vos formulaires.

## Conditions préalables

Pour tirer le meilleur parti de ce tutoriel, nous vous recommandons de respecter les conditions préalables suivantes :

* Expérience avec AEM Forms as a Cloud Service
* Accès aux balises Experience Platform
* Accès à Adobe Analytics

Ce tutoriel utilise un formulaire adaptatif simple intégré à AEM Forms et mesure les envois de formulaire pour les valeurs d’état de résidence, ainsi que les champs qui génèrent des erreurs de validation.
