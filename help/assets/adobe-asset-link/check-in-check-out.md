# Archivage et extraction {#check-in-check-out}

Adobe Asset vous permet d’extraire AEM Assets directement à partir de l’application créative sur laquelle vous travaillez et de commencer immédiatement à apporter des modifications. Vous pouvez facilement archiver les ressources dans AEM Assets avec un contrôle de version et des commentaires appropriés une fois que vous avez fini de les modifier dans votre application de Creative Cloud.
