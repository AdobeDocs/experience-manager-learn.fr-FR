# Découvrez les avantages de Dynamic Media pour les images {#snapshot}

Dynamic Media Snapshot est un outil de démonstration visuel conçu pour illustrer la puissance de Dynamic Media pour une diffusion d’images optimisée et dynamique. Testez des images de test ou des URL Dynamic Media afin d’observer visuellement la sortie de divers modificateurs d’image Dynamic Media et d’optimiser l’imagerie dynamique pour la taille de fichier (avec diffusion WebP et AVIF), le rapport de pixels réseau et de périphérique (RPD).
